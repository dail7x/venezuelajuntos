# Análisis Impeccable de UI y Copy

## Veredicto Anti AI Slop

Estado actual: riesgo medio-alto. La plataforma ya resuelve flujos importantes, pero el primer contacto todavía suena demasiado genérico y operativo para una tragedia nacional. Frases como "base operativa pública", "peticiones de ayuda ciudadana" o "centraliza reportes" explican el sistema, pero no acompañan a una persona que busca a alguien o intenta avisar que está con vida.

## Hallazgos Prioritarios

**P0: tono emocional insuficiente en el hero**
El hero debe reconocer el momento histórico y humano antes de describir funcionalidades. La prioridad no es vender una plataforma, sino ayudar a encontrar personas, registrar información confiable y acompañar.

**P1: demasiadas acciones al mismo nivel**
Hay seis CTAs principales. Es funcional, pero genera carga cognitiva. La jerarquía debe ordenar primero: buscar, reportar desaparecido, avisar encontrado, pedir ayuda, ofrecer ayuda.

**P1: lenguaje tecnico o institucional**
Términos como "base operativa", "deduplicación", "match", "coordinación" y "fuente pública" pueden sentirse fríos o internos. Deben cambiarse por lenguaje ciudadano: "revisar posibles repetidos", "cruzar información", "equipo de verificación".

**P2: microcopy de formularios puede sonar exigente**
Frases como "necesitamos coordenadas exactas" deben suavizarse: pedir colaboración, explicar por qué ayuda y recordar que no todos podrán compartir ubicación.

**P2: avisos legales correctos pero poco humanos**
El disclaimer es necesario, pero debe decir con tacto que esta es una herramienta ciudadana, construida para acompañar reportes de familias, amistades y voluntarios, sin sustituir servicios oficiales.

Total: 27/40. Buena base funcional, necesita dirección de voz y jerarquía.

## Dirección de Copy

La voz recomendada es: sobria, venezolana sin localismos excesivos, directa y compasiva. Evitar dramatizar. Evitar lenguaje de marketing. Usar verbos simples: buscar, reportar, avisar, ayudar, confirmar, cuidar.

Ejemplo de tono:
"Cada reporte puede ayudar a una familia a encontrar una respuesta. Comparte sólo información que puedas sostener y evita publicar datos sensibles innecesarios."
